Introduction to Database Systems

Advanced DBMS • Unit 1: Relational Data Models

A database-management system (DBMS) is a collection of interrelated data and a set of programs to access those data. The collection of data, usually referred to as the database, contains information relevant to an enterprise.

Key Concepts:

  • Data Independence
  • Efficient Data Access
  • Data Integrity and Security
  • Concurrent Access and Crash Recovery

Fig 1.1: 3-Schema Architecture Overview

The primary goal of a DBMS is to provide a way to store and retrieve database information that is both convenient and efficient. High-level data models allow users to perceive data as abstractions rather than complex low-level bit structures.
